# WiFi Pineapple Payload

This repository contains a WiFi Pineapple payload.


## Video Overview

A short video overview and review of this payload is available on YouTube: COMING SOON


## Description

USB auto backups Payload will mount a usb flash drive that is plugged into the pager then let you select from folders on the device to zip them up and copy over to the usb flash drive. Once completed it will safely unmount usb storage on the WiFi PineApple Pager.
